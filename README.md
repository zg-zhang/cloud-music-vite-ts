# cloud-music-vite-ts
 
> 感谢 `https://github.com/Binaryify/NeteaseCloudMusicApi` 提供了后台服务 <br />

* 技术栈: TypeScript, React, immer, use-immer, unstated-next, styled-components ...

## 项目目录结构

```text
├─ assets       图片字体资源
├─ components   组件
├─ config       配置
├─ constants    常量
├─ containers   状态容器
├─ events       事件 
├─ layouts      布局
├─ pages        页面
├─ routes       路由  
├─ services     数据层
├─ styles       样式
├─ tools        工具
├─ typings      类型定义
├─ App.tsx      根组件
└─ main.tsx     入口文件
```
